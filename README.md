# CineSnap - Smart Log Video Viewer & Snapshot Tool

CineSnap is a specialized tool for video creators who shoot in Log (S-Log3, Apple Log, etc.). It acts as a lightweight, intelligent viewer that instantly normalizes flat footage for preview, comparison, and snapshots.

## Project Structure

This repository contains the CineSnap project and its related assets.

- **`lut-viewer-app/`**: **Main Application Code**. Go here to install, build, and run the Electron app.
  - [View App README](./lut-viewer-app/README.md)
  - [Installation Guide](./lut-viewer-app/README.md#installation-for-collaborators)
  - [App Store Guide](./lut-viewer-app/MACOS_PUBLISHING.md)

- **`marketing/`**: Assets, copy, and materials for marketing CineSnap.
- **`reference/`**: Documentation, competitive analysis, and reference materials.

## Getting Started

To get started with development or to build the app, navigate to the `lut-viewer-app` directory:

```bash
cd lut-viewer-app
npm install
npm run dev
```

See [lut-viewer-app/README.md](./lut-viewer-app/README.md) for full documentation.
